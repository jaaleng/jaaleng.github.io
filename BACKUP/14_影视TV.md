# [影视TV](https://github.com/jaaleng/jaaleng.github.io/issues/14)

一个电视直播、电影、剧集集成的app，有电视和手机版本。

在wx有公号：OK猫开发。

仓库地址： [Github](https://github.com/FongMi/Release/tree/main/apk/release)

**影視TV**

```
mobile = 手機版
leanback = 電視版

java = jar + js 爬蟲
python = jar + js + py 爬蟲

arm64_v8a = 64 位元
armeabi_v7a = 32 位元

dev = 內測版
kitkat = 4.x 版
release = 正式版
```

公号里也能下载到app的。

内部接口：http://ok321.top/tv
