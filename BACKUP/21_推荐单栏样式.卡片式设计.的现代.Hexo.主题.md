正如其名，「Cards」，一款 单栏样式 & 卡片式设计 的现代 Hexo 主题。

![](https://pic.imgdb.cn/item/66ace814d9c307b7e97faaea.png)

Installation 安装

```
cd hexo
git clone https://github.com/ChrAlpha/hexo-theme-cards.git themes/cards

vi _config.yml
-theme: landscape
+theme: cards

hexo s --debug
```
