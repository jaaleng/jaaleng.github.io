# [索尼对《星鸣特攻复活 Mod 采取版权打击行动](https://github.com/jaaleng/jaaleng.github.io/issues/256)

《星鸣特攻》作为游戏史上最著名的失败作品之一，在粉丝开发复活 Mod 后再次引发关注。该游戏此前耗费数亿美元开发，但仅运营两周就被索尼关闭。

近日，一个小团队花费数月时间开发了自定义服务器软件，让玩家能够重新体验完整的游戏对战。

![](https://pic.imgdd.cc/item/691adb17c828c4c6def6d0c5.jpg)


然而好景不长，索尼已开始对相关游戏录像发出 DMCA 版权删除通知。

开发者 Red 在 Discord 上表示项目仍在进行中但存在 Bug，并强调不会为没有游戏文件的用户提供盗版内容，希望项目能在法律允许范围内继续进行。

目前 Mod 服务器仍在运行，但前景堪忧。
