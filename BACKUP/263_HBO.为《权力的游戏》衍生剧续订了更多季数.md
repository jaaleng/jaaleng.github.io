# [HBO 为《权力的游戏》衍生剧续订了更多季数](https://github.com/jaaleng/jaaleng.github.io/issues/263)

[HBO 为《权力的游戏》衍生剧续订了更多季数](https://www.theverge.com/streaming/825136/hbo-renewals-house-of-the-dragon-a-knight-of-seven-kingdoms-i-love-la)

![](https://pic.imgdd.cc/item/691fdd4ac828c4c6defac20c.jpg)

HBO 最新的《权力的游戏》前传系列还未播出，但已经计划至少再续播一季。

今天，HBO 宣布《七王国的骑士》和《龙之家族》已分别续订第二季和第四季。

除了《权力的游戏》的前传剧集外，HBO 还订购了 Tim Robinson 的《椅子公司》、Rachel Sennott 的《我爱洛杉矶》和 Mark Ruffalo 主演的《特遣任务》的更多剧集。
