# [10 个真正改变工作流的 Claude Skills](https://github.com/jaaleng/jaaleng.github.io/issues/280)

Skills 上线一个月了.说实话，有些是凑数的，但有些是真·硬核。


1. Rube MCP Connector 
这个太炸了。通过这一个服务器就能让 Claude 连接 Slack、GitHub、Notion 等 500 多个应用，不用一个个单独搞授权。搞自动化的兄弟们，这玩意儿能省下你半条命。

2. Superpowers 
obra 大神开发的工具包。/brainstorm、/write-plan、/execute-plan 三连击，直接把 Claude 从聊天机器人变成正经的开发工作流。

3. Document Suite 
官方出品。让 Claude 真正精通 Office 全家桶。不是简单的“读”，而是能带格式、带公式地生成 Word/Excel/PPT/PDF。Pro 用户直接内置.

4. Theme Factory 
上传一次品牌规范，Claude 生成的所有素材（Artifacts）自动对齐你的配色和字体。
5. Algorithmic Art 
用自然语言写 p5.js 生成艺术。“蓝紫渐变流场，5000 粒子，随机种子 42”，Boom，直接出可复现的艺术图。

6. Slack GIF Creator 
别去 Giphy 搜图了，直接告诉 Claude 你想要啥，生成 Slack 专用动图

7. Webapp Testing 
Playwright 自动化。直接跟 Claude 说“测一下登录流程”，它自动写脚本 + 跑测试。

8. MCP Builder 
自动生成 MCP 服务器模版。

9. Brand Guidelines 
跟 Theme Factory 类似，但支持多品牌管理。多项目之间一键切换.

10. Systematic Debugging 
让 Claude 秒变资深架构师。根本原因分析 → 假设 → 修复 → 文档，一套连招。
