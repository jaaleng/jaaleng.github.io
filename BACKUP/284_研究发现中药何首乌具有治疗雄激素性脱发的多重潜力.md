# [研究发现中药何首乌具有治疗雄激素性脱发的多重潜力](https://github.com/jaaleng/jaaleng.github.io/issues/284)

发表在《全人整合药学杂志》（Journal of Holistic Integrative Pharmacy）的一项研究综述显示，传统中药何首乌（Polygonum multiflorum）在治疗雄激素性脱发（AGA）方面展现出多重生物学机制。


研究指出，何首乌能同时抑制双氢睾酮、保护毛囊、激活 Wnt 与 Shh 信号通路并改善头皮血液循环。

![IMG_20260211_085630_882.jpg](https://i.829259.xyz/api/rfile/IMG_20260211_085630_882.jpg)


该研究对比了非那雄胺与米诺地尔等现行药物，强调何首乌在经正确炮制后安全性更高，且具备促进毛发再生的潜力。目前，研究人员呼吁开展更多高质量临床试验以进一步验证其疗效并确定最佳使用方案。
