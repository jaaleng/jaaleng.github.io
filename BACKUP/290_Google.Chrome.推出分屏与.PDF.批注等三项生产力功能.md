# [Google Chrome 推出分屏与 PDF 批注等三项生产力功能](https://github.com/jaaleng/jaaleng.github.io/issues/290)

Google 宣布为 Chrome 浏览器推出三项生产力新功能，涵盖分屏模式（Split view）、PDF 批注及直接保存至 Google Drive。

<!-- more -->

![IMG_20260220_140944_943.jpg](https://i.829259.xyz/api/rfile/IMG_20260220_140944_943.jpg)

分屏模式支持在同一窗口并行操作两个标签页，以减少频繁切换。PDF 批注功能允许用户直接在浏览器内高亮文本并添加笔记，无需下载第三方应用。

![IMG_20260220_140929_076.jpg](https://i.829259.xyz/api/rfile/IMG_20260220_140929_076.jpg)

此外，用户可将 PDF 直接存入 Google Drive 的指定文件夹。上述功能旨在优化多任务处理流程，提升办公与协作效率。

[google blog](https://blog.google/products-and-platforms/products/chrome/chrome-productivity-improvements/)
