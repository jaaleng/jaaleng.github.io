# [Telegram 终于解决了中文搜索的问题](https://github.com/jaaleng/jaaleng.github.io/issues/302)

Telegram 终于解决了中文搜索的问题

为什么以前的搜索那么“残废”？

Telegram 长期以来在中文搜索上有一个顽疾：由于缺乏分词机制，必须输入完整句子或特定分词（空格、标点或符号）才能搜到内容。

这个问题的原因其实很简单：英文有空格分词，中文没有。在英文里，单词之间有天然空格，搜索非常精准。但中文是一句话连在一起写的，系统不容易判断一个词从哪里开始、到哪里结束。这个问题在中、日、韩等语言里都存在。


搜索的“进化时刻”：

在最近的版本更新中，Telegram 从服务器端到底层算法做了重构，正式引入了针对中日韩语言的索引机制。简单说，系统现在具备了“子字符串匹配”的能力——即使你只搜某个词组中的一部分，系统也能在长句里精准找到相关内容。

实际体验改善很明显：中文搜索比之前好用了。而且这次更新不需要更新 App，直接搜索就行。之前的历史消息也被重新索引了，同样可以正常搜索。


一个小提示：

目前搜索时可能会有一个小情况：比如你搜“飞起”，结果里可能会出现“起飞”、“要起飞”、“起飞了”等相关内容。这是因为系统做了智能匹配。

如果你想精确搜索某个词的内容，可以给词语加上英文单引号，比如搜：'飞起'，这样结果就会更准确。
