# [《自己动手写 Git》中文翻译](https://github.com/jaaleng/jaaleng.github.io/issues/98)

 ### 介绍
 本教程将介绍如何使用 Python 从头开始实现简化版 Git，目的是通过构建一个包含 init、add、commit 和 log 等基本命令的基本版 
 本，帮助读者理解 Git 的核心原理。本教程只需了解一些基本的 Git、Python 和 shell 知识即可学习。
